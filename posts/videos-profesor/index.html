2023-11-06 Actualizado 2023-12-15 1 minutos lectura

Programas útiles

  • Carnac: Para mostrar atajos de teclado en pantalla.
  • OBS: Para grabar.
  • KDEnlive: Para editar vídeos.
  • ZoomIt: Para hacer zoom y pintar en la pantalla.
  • Mouse utilities de PowerToys: Para mostrar los clicks del ratón.
  • Canva: Para grabar y editar vídeos.
